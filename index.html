Full Stack Developer

My name is Cleveland Williams and I am a web developer who specializes in creating dynamic and beautiful websites. Before my journey as a web developer I repaired computers & build high end gaming pcs. After building computers for so long, I became interested in web development after which I encountered valuable resources. Among them were CodingDojo an onsite bootcamp, Udemy and FreeCodeCamp.

Process Of Creation

Building Blocks

Planning

Create an outline of the site content along with a functional flow chart. Document programmatic flow and gather general information including samples, style guides, images and graphic files.

Design

Create color layouts based on approved ‘wireframe’ layouts. Research stock photography and develop graphics. Create style guide of colors and fonts for site development.

Content Writing

Content writing and compiling usually overlaps with other stages of website creation, and its role can’t be underestimated. Content writing involves creation of headlines, text editing, writing new text, compiling the existing text, etc...

Coding

The home page is usually created first, and then all sub-pages are added, according to the website hierarchy that was previously created in the form of a sitemap. All static page elements that were designed during the layout creation will be created and tested.
